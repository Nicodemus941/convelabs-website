# ConveLabs Website

Luxury mobile phlebotomy and lab services website.

## Tech Stack

- React 18 + TypeScript
- Vite 5
- Tailwind CSS + shadcn/ui
- Supabase (Auth, Database, Edge Functions)
- Stripe Payments
- Deployed on Vercel

## Getting Started

```bash
npm install
npm run dev
```

## Environment Variables

Copy `.env.example` to `.env` and fill in your Supabase credentials:

```bash
cp .env.example .env
```

## Build

```bash
npm run build
npm run preview
```
